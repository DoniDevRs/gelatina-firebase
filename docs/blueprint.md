# **App Name**: Gelatina Glow

## Core Features:

- Landing Page Display: Display a modern, mobile-first landing page with a feminine and premium aesthetic, highlighting the benefits of the Gelatina Anti-Caneta method.
- Hero Section Design: Implement a visually appealing hero section with a clear headline, subheadline, image, and call-to-action button to capture user attention immediately.
- How It Works Section: Show the benefits of the 'Gelatina Anti-Caneta' method.
- Benefits Showcase: Design elegant cards to showcase the benefits of the Gelatina Anti-Caneta method, such as natural hunger reduction, prolonged satiety, and reduced late-night cravings.
- Social Proof Display: Show elegant card-based testimonials in a modern design.
- Offer Presentation: Show central premium card-based method and its inclusions (Exclusive App, 14-Day Protocol, Saciedade Strategy, Practical Guide in PDF, Immediate Access), using highlight-price features and CTA button.
- FAQ Section: Show question answering in the accordion section with the most important questions from customers, aiming on design minimalism

## Style Guidelines:

- Primary color: Dusty rose (#D65A9A) to evoke a feminine and sophisticated feel.
- Background color: Light pink (#F8E9F1) to create a clean and airy backdrop.
- Accent color: Darker rose (#B83280) to highlight key elements and calls to action.
- Headline font: 'Playfair' serif for headlines to convey elegance and sophistication. Note: currently only Google Fonts are supported.
- Subtitles font: 'Poppins' sans-serif for subtitles for a clear and elegant touch. Note: currently only Google Fonts are supported.
- Body font: 'Inter' sans-serif for body text, optimizing it for better legibility and visual organization. Note: currently only Google Fonts are supported.
- Mobile-first approach with generous spacing and visual breathing room for a clean, uncluttered design.
- Use elegant, custom icons in a delicate pink shade to complement the overall aesthetic.
- Implement subtle hover effects on buttons and cards to enhance user engagement and provide a premium feel.
